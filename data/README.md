##  frontline_workers.csv  
Simulated dataset representing frontline public service workers.  
It reproduces the structure of the ENQSP staff module, including identification fields, PPSP information, agent characteristics, QA indicators (QA21–QA43), and sector metadata.

##  decision_makers.csv  
Simulated dataset representing institutional decision-makers.  
It mirrors the ENQSP decision-maker module with perception indicators (Q1–Q9), sector-level information, satisfaction tracking (Q81–Q82), recommendations, and system metadata.
